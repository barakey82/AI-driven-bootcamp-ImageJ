12 - PyImageJ ‚Äî Fiji Meets Python

#python #pyimagej #advanced #pipeline

Overview

PyImageJ lets you run the full ImageJ/Fiji ecosystem from within Python. This means you can combine Python's AI libraries (PyTorch, TensorFlow, Cellpose) with ImageJ's established image processing tools in a single script.

üîó Documentation: https://pyimagej.readthedocs.io
üîó GitHub: https://github.com/imagej/pyimagej

Installation

pip install pyimagej
# For full Fiji functionality:
pip install pyimagej[all]

Quick Start

import imagej

# Initialize with Fiji (first run downloads ~1GB)
ij = imagej.init('sc.fiji:fiji')

# Open an image
img = ij.io().open('my_image.tif')

# Run any ImageJ command
ij.py.run_macro('run("Gaussian Blur...", "sigma=2")')

# Convert between numpy and ImageJ
import numpy as np
np_array = ij.py.from_java(img)     # ImageJ ‚Üí numpy
java_img = ij.py.to_java(np_array)   # numpy ‚Üí ImageJ

Key Use Cases

Use Case 1: Pre-process in ImageJ, segment in Python

# Use ImageJ's rolling ball background subtraction
ij.py.run_macro('run("Subtract Background...", "rolling=50")')
np_img = ij.py.from_java(ij.py.active_dataset())

# Then run Cellpose
from cellpose import models
model = models.Cellpose(model_type='cyto3')
masks, _, _, _ = model.eval(np_img, diameter=30, channels=[0,0])

Use Case 2: Segment in Python, measure in ImageJ

# After Cellpose segmentation, use ImageJ for measurements
java_masks = ij.py.to_java(masks)
ij.py.run_macro("""
    run("Set Measurements...", "area mean perimeter shape redirect=None");
    run("Analyze Particles...", "display clear");
""")

Use Case 3: Use ImageJ file format support

# ImageJ can read proprietary formats (Leica LIF, Zeiss CZI, Nikon ND2)
# via Bio-Formats
img = ij.io().open('experiment.lif')
