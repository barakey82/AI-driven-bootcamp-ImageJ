Overview

Micro-SAM adapts Meta's Segment Anything Model (SAM) specifically for microscopy. SAM can segment any object in an image with minimal user input â€” just click on it or draw a bounding box. Micro-SAM fine-tunes this capability for biological images.

ðŸ”— GitHub: https://github.com/computational-cell-analytics/micro-sam
ðŸ”— Paper: Archit et al., "Segment Anything for Microscopy" (Nature Methods, 2024)

What Makes SAM Special

Traditional AI segmentation: Train a specific model for a specific task.
SAM: One model that can segment almost anything, guided by simple prompts.

Prompt types:
- Point: Click on an object â†’ SAM segments it
- Box: Draw a bounding box â†’ SAM segments everything inside
- Automatic: SAM finds and segments all objects without prompts

Installation

pip install micro_sam

Using Micro-SAM with napari

# Launch the napari-based GUI
micro_sam.sam_annotator

Interactive Segmentation

  1. Load your microscopy image
  2. Click on a cell â†’ Micro-SAM segments it
  3. Click on more cells to add to your selection
  4. Accept or refine the segmentation
  5. Export masks

Automatic Segmentation

from micro_sam.automatic_segmentation import automatic_instance_segmentation

masks = automatic_instance_segmentation(
    image,
    model_type="vit_b_lm",  # Micro-SAM model fine-tuned for light microscopy
)

Available Micro-SAM Models

Model Trained On Best For
vit_b_lm Light microscopy Fluorescence, phase contrast, DIC
vit_b_em_organelles Electron microscopy Organelle segmentation in EM
vit_b_em_boundaries Electron microscopy Membrane/boundary detection

When to Use Micro-SAM vs. StarDist/Cellpose

Scenario Best Tool
Batch processing thousands of images StarDist / Cellpose (fully automatic)
Interactive annotation / correction Micro-SAM (click-to-segment)
Novel object types with no existing model Micro-SAM (zero-shot segmentation)
Creating training data for other models Micro-SAM (fast annotation)

Micro-SAM is especially powerful as an annotation tool â€” use it to quickly create training data, then train a specialized model (StarDist, Cellpose) for batch processing.
