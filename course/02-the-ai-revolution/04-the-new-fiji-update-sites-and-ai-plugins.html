04 - The New Fiji â€” Update Sites and AI Plugins

#fiji #installation #deep-learning

Overview

The Fiji you remember is still there â€” but it now has an ecosystem of AI plugins that transform it into a deep learning workstation. All of these are installed through the Update Sites mechanism you may already know.

Updating Your Fiji

Step 1: Get the Latest Fiji

If your Fiji installation is more than a year old, consider downloading a fresh copy:

ğŸ”— Download Fiji: https://fiji.sc/#download

Choose the version for your OS. Fiji is self-contained â€” just unzip and run.

Step 2: Update Everything

  1. Open Fiji
  2. Go to Help â†’ Update...
  3. Let it check for updates
  4. Click Apply Changes
  5. Restart Fiji

Step 3: Enable AI Update Sites

This is where the magic happens:

  1. Go to Help â†’ Update...
  2. Click Manage Update Sites
  3. Find and check these boxes:
Update Site What It Provides Priority
CSBDeep Deep learning infrastructure, CARE denoising Essential
StarDist Star-convex object detection for nuclei Essential
DeepImageJ Run BioImage Model Zoo models Essential
CLIJ GPU-accelerated image processing Recommended
CLIJ2 Updated GPU acceleration library Recommended
TrackMate-StarDist StarDist detector for TrackMate Recommended
LabKit Interactive pixel classification Optional
MorphoLibJ Advanced morphological operations Optional
  1. Click Close
  2. Click Apply Changes
  3. Restart Fiji
Java version matters
CSBDeep and StarDist require Java 8 or higher. If you downloaded Fiji recently, this should be fine. If you have an ancient installation, update Fiji first.

Verifying Your Installation

After restarting, verify each plugin is available:

StarDist

  • Go to Plugins â†’ StarDist
  • You should see StarDist 2D and optionally StarDist 3D
  • If you see an error about TensorFlow, see the Troubleshooting FAQ

DeepImageJ

  • Go to Plugins â†’ DeepImageJ
  • You should see DeepImageJ Run, DeepImageJ Install Model, etc.

CLIJ2

  • Go to Plugins â†’ ImageJ on GPU (CLIJ2)
  • You should see the CLIJ2 menu with many operations

TrackMate

  • Go to Plugins â†’ Tracking â†’ TrackMate
  • In the detector selection step, you should see StarDist detector as an option

Installing Python-Based Tools (Optional)

Some tools require a Python environment. This is optional but opens up significantly more capabilities.

Cellpose

Cellpose runs in Python and integrates with Fiji via a wrapper:

# Install Cellpose (requires Python 3.8+)
pip install cellpose

# Or with GPU support (NVIDIA GPU + CUDA required)
pip install cellpose[gui]

# Verify installation
python -c "from cellpose import models; print('Cellpose OK')"

ğŸ”— Cellpose installation guide: https://cellpose.readthedocs.io/en/latest/installation.html

PyImageJ (for Python integration)

# Install PyImageJ
pip install pyimagej

# Also install dependencies
pip install numpy scikit-image pandas matplotlib

# Verify
python -c "import imagej; print('PyImageJ OK')"

ğŸ”— PyImageJ documentation: https://pyimagej.readthedocs.io/en/latest/

Micro-SAM

pip install micro_sam

# Verify
python -c "import micro_sam; print('Micro-SAM OK')"

ğŸ”— Micro-SAM GitHub: https://github.com/computational-cell-analytics/micro-sam
