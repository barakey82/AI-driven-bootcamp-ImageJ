The BioImage Model Zoo

What It Is

Think of it as an "app store" for bioimage AI models. Researchers train models, package them in a standard format, and share them for anyone to use.

How to Browse

  1. Go to https://bioimage.io
  2. Browse by task: segmentation, denoising, detection, classification
  3. Browse by modality: fluorescence, electron microscopy, histology, phase contrast
  4. Each model has:
  5. Description of what it does
  6. What data it was trained on
  7. Test images to verify it works
  8. Download link

Using DeepImageJ in Fiji

Installing a Model

Method 1: From the Zoo (recommended)
1. Go to https://bioimage.io
2. Find a model that matches your data
3. Download the model (.zip file)
4. In Fiji: Plugins â†’ DeepImageJ â†’ Install Model
5. Select the downloaded .zip file
6. The model is now available in DeepImageJ

Method 2: From within Fiji
1. Plugins â†’ DeepImageJ â†’ DeepImageJ Install Model
2. Browse available models directly from Fiji
3. Click Install

Running a Model

  1. Open your image in Fiji
  2. Plugins â†’ DeepImageJ â†’ DeepImageJ Run
  3. Select the installed model from the dropdown
  4. Adjust pre/post-processing if needed
  5. Click Run
  6. Results appear as a new image (semantic segmentation mask, denoised image, etc.)

Converting Results to ROIs

DeepImageJ outputs typically need post-processing to become usable ROIs:

// After running DeepImageJ, convert label image to ROIs
// Assuming the output is a label image where each object has a unique value

// Threshold each label to create individual ROIs
selectWindow("DeepImageJ_output");
run("Duplicate...", "title=labels");

nLabels = getValue("Max");
for (i = 1; i <= nLabels; i++) {
    selectWindow("labels");
    run("Duplicate...", "title=temp");
    setThreshold(i, i);
    run("Create Selection");
    roiManager("Add");
    close("temp");
}

// Now measure
roiManager("Measure");

The Model Format: BioImage.IO Model Resource Description

Models in the zoo use a standardized format (RDF â€” Resource Description Format) that includes:

# Example model.yaml (simplified)
name: "Nucleus Segmentation"
description: "StarDist model for fluorescent nuclei"
authors:
  - name: "Author Name"
type: model
inputs:
  - name: input
    axes: "bxyc"        # batch, x, y, channel
    data_type: float32
    shape: [1, 256, 256, 1]
outputs:
  - name: output
    axes: "bxyc"
    data_type: float32
weights:
  tensorflow_saved_model_bundle:
    source: ./weights
test_inputs:
  - test_input.npy
test_outputs:
  - test_output.npy

This standardization means the same model can run in:
- Fiji (via DeepImageJ)
- Python (via bioimageio.core)
- napari
- Icy
- QuPath
- ImJoy

Building Your Own Pipeline with BioImage Zoo Models

Step 1: Find a Relevant Model

Search https://bioimage.io with keywords matching your data:
- "nuclei fluorescence"
- "mitochondria electron microscopy"
- "cell membrane confocal"
- "denoising"

Step 2: Test on Your Data

Download the model, run it on a few of your images, and evaluate:
- Do the segmentation boundaries look correct?
- Are objects being missed or falsely detected?
- Does it handle the full range of conditions in your dataset?

Step 3: Decide Next Steps

Evaluation Result Next Step
Works perfectly Use it! You're done.
Works mostly (>80% accuracy) Fine-tune with a few annotated images
Doesn't work well Try a different model, or train from scratch
Nothing relevant in the zoo Train your own model (see 13 - ZeroCostDL4Mic - Train Models for Free)

Advanced: Using Models in Python with bioimageio.core

import bioimageio.core
from bioimageio.core import load_resource_description
import numpy as np
from skimage import io

# Load model from the zoo
model = load_resource_description("<a href="https://bioimage.io/model/...">https://bioimage.io/model/...</a>")

# Or from a local file
model = load_resource_description("path/to/model.zip")

# Create prediction pipeline
pipeline = bioimageio.core.create_prediction_pipeline(model)

# Run inference
input_image = io.imread("my_image.tif").astype(np.float32)
# Reshape to match model expectations (batch, x, y, channel)
input_tensor = input_image[np.newaxis, ..., np.newaxis]

result = pipeline(input_tensor)
output = result[0].squeeze()  # Remove batch and channel dims

io.imsave("segmentation.tif", output)
