Tool Installation Guide

#installation #setup #reference

Prerequisites

Requirement Minimum Recommended
Fiji Latest version Latest version
Java 8 11+
Python 3.8 3.10+
RAM 8 GB 16+ GB
GPU Not required NVIDIA with CUDA (for training)
Disk Space 5 GB 20+ GB (for models and data)

Step 1: Install/Update Fiji

Fresh Installation

  1. Download from https://fiji.sc/#download
  2. Extract to a location with write permissions (NOT Program Files on Windows)
  3. Run the Fiji executable

Update Existing Installation

  1. Open Fiji
  2. Help → Update...
  3. Apply Changes
  4. Restart

Step 2: Enable Update Sites

  1. Help → Update...
  2. Click Manage Update Sites
  3. Enable the following (check the box):

Essential

Recommended

Optional

  • [ ] 3D ImageJ Suite — 3D analysis tools

  • Click Close → Apply Changes → Restart Fiji

Step 3: Install Python Tools

Option A: Using pip (simplest)

# Create a virtual environment (recommended)
python -m venv bioimage_env
# Activate it:
#   Windows: bioimage_env\Scripts\activate
#   Mac/Linux: source bioimage_env/bin/activate

# Install core packages
pip install cellpose
pip install pyimagej
pip install micro_sam
pip install numpy scikit-image pandas matplotlib
pip install n2v  # Noise2Void

Option B: Using conda (more reliable for complex dependencies)

# Create environment
conda create -n bioimage python=3.10
conda activate bioimage

# Install packages
conda install -c conda-forge pyimagej openjdk=11
pip install cellpose
pip install micro_sam
conda install -c conda-forge numpy scikit-image pandas matplotlib

Step 4: Verify Everything Works

Test Fiji Plugins

Open Fiji and verify each plugin loads:

Plugins → StarDist → StarDist 2D                    (should open dialog)
Plugins → DeepImageJ → DeepImageJ Run               (should open dialog)
Plugins → ImageJ on GPU (CLIJ2)                      (should show menu)
Plugins → Tracking → TrackMate                       (should start wizard)

Test Python

python -c "from cellpose import models; print('✓ Cellpose')"
python -c "import imagej; print('✓ PyImageJ')"
python -c "import numpy; print('✓ NumPy')"
python -c "import skimage; print('✓ scikit-image')"
python -c "import pandas; print('✓ pandas')"

Troubleshooting

"TensorFlow not found" when running StarDist

  • Make sure CSBDeep update site is enabled
  • Restart Fiji after enabling update sites
  • Try: Help → Update... → Apply Changes again

Fiji runs out of memory

  • Edit → Options → Memory & Threads → increase maximum memory
  • Set to ~75% of your system RAM

Cellpose is slow

  • CPU mode works but is slower
  • For GPU: install pip install cellpose[gui] with NVIDIA CUDA toolkit
  • GPU is mainly needed for training, not inference

PyImageJ won't initialize

  • Ensure Java 11+ is installed
  • Try: ij = imagej.init('sc.fiji:fiji', mode='headless')
  • Check: https://pyimagej.readthedocs.io/en/latest/Troubleshooting.html

DeepImageJ model won't load

  • Ensure the model format matches your DeepImageJ version
  • Check Java memory allocation
  • Try re-downloading the model
