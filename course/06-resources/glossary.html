Glossary

#reference #glossary

A

Analyze Particles
Classical Fiji command that counts and measures objects in binary images. Replaced by AI segmentation tools for complex cases.

Augmentation (Data Augmentation)
Artificially expanding training data by applying transformations (rotation, flipping, scaling, intensity changes). Helps models generalize better.

B

BioImage Model Zoo
Community repository of pre-trained deep learning models for bioimage analysis, accessible at https://bioimage.io.

Binary Image / Mask
Image with only two values (0 and 1, or 0 and 255) representing background and foreground.

C

CARE (Content-Aware image REstoration)
Deep learning method for restoring degraded microscopy images while preserving biological structure.

Cellpose
Generalist deep learning algorithm for cell segmentation, capable of handling irregular cell shapes.

CLIJ / CLIJ2
Fiji plugins that run image processing operations on the GPU for dramatic speed improvements.

CNN (Convolutional Neural Network)
Type of neural network that processes images using learnable filters. Foundation of most bioimage AI.

CSBDeep
Fiji plugin providing the infrastructure for running deep learning models (CARE, StarDist) inside Fiji.

D

DeepImageJ
Fiji plugin for running BioImage Model Zoo models inside ImageJ.

Deep Learning
Subset of machine learning using neural networks with multiple layers. Learns features directly from data rather than using hand-crafted rules.

E

Epoch
One complete pass through the entire training dataset during model training.

F

Fine-tuning
Taking a pre-trained model and further training it on your specific data. Requires less data than training from scratch.

Flow Field
The approach Cellpose uses for segmentation — predicting directional "flows" that point toward cell centers.

G

GPU (Graphics Processing Unit)
Hardware originally designed for graphics rendering, now used for accelerating deep learning and image processing.

Ground Truth
The correct answer (e.g., manually annotated segmentation) used to train or evaluate a model.

I

Inference
Running a trained model on new data to get predictions (as opposed to training).

Instance Segmentation
Segmentation that distinguishes individual objects (nucleus #1 vs. nucleus #2), not just classes.

L

Label Image / Label Map
Image where each pixel contains an integer identifying which object it belongs to (0 = background, 1 = object 1, 2 = object 2, etc.).

LabKit
Fiji plugin for interactive machine learning segmentation using painted annotations.

LAP Tracker (Linear Assignment Problem)
Tracking algorithm in TrackMate that links detections across frames by minimizing total displacement.

LoG (Laplacian of Gaussian)
Classical blob detection method used in TrackMate. Detects round bright objects.

M

Micro-SAM
Adaptation of Meta's Segment Anything Model (SAM) for microscopy images.

Model
A trained neural network that can make predictions on new data.

N

NMS (Non-Maximum Suppression)
Post-processing step that removes duplicate detections for the same object. Controlled by the "NMS threshold" or "overlap threshold" parameter.

Noise2Void (N2V)
Self-supervised denoising method that trains on noisy images without requiring clean references.

O

Otsu Threshold
Classical automatic thresholding method that finds the optimal threshold by minimizing inter-class variance.

P

Pre-trained Model
A model already trained on a large dataset, ready to use without additional training.

PyImageJ
Python library that allows running ImageJ/Fiji functions from within Python.

R

Random Forest
Machine learning algorithm that combines many decision trees. Used by LabKit and Ilastik for pixel classification.

ROI (Region of Interest)
A defined area in an image, typically representing a detected object. Managed in Fiji's ROI Manager.

S

SAM (Segment Anything Model)
Foundation model by Meta that can segment any object in an image with minimal prompts.

Semantic Segmentation
Segmentation that labels pixels by class (e.g., "nucleus" vs. "background") without distinguishing individual objects.

StarDist
Deep learning method for detecting and segmenting star-convex objects (especially nuclei) in microscopy images.

T

Threshold
Converting a grayscale image to binary by choosing an intensity cutoff. Pixels above = foreground, below = background.

TrackMate
Fiji plugin for tracking objects (cells, particles) across time-lapse sequences.

Training
The process of adjusting a model's parameters to learn from annotated examples.

U

U-Net
Neural network architecture widely used for biomedical image segmentation. Named for its U-shaped encode-decode structure.

Update Site
Fiji's mechanism for installing and updating plugins from community repositories.

W

Watershed
Classical algorithm for separating touching objects using a distance transform. Often fails on complex arrangements.

Weights
The numerical parameters inside a neural network that are adjusted during training.

Z

ZeroCostDL4Mic
Collection of Google Colab notebooks for training deep learning models for microscopy without local hardware or cost.
