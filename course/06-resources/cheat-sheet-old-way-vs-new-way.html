Cheat Sheet — Old Way vs New Way

#reference #cheat-sheet #comparison

Quick Reference Card

Print this or keep it on your desk. For each common task, the classical approach and the modern AI approach side by side.

Segmentation

Nuclei in Fluorescence

Old Way New Way
Method Threshold → Watershed → Analyze Particles StarDist 2D
Steps ~10 (threshold, mask, fill, watershed, analyze) ~3 (open, run StarDist, measure)
Accuracy 60-80% on dense tissue 90-98%
Touching objects Frequently fails Handles well
Time per image 5-30 sec + manual correction 2-10 sec, minimal correction
Fiji command run("Watershed") → run("Analyze Particles...") Plugins → StarDist → StarDist 2D

Irregular Cell Shapes

Old Way New Way
Method Manual ROI drawing Cellpose (cyto3)
Steps Draw each cell by hand Set diameter → run
Time per image 10 min - 2 hours 5-30 seconds
Scalability Not feasible for large datasets Batch-processable
Command Manual python -m cellpose --dir ./images --pretrained_model cyto3

Organelles in EM

Old Way New Way
Method Manual tracing DeepImageJ + BioImage Zoo model
Steps Trace each organelle by hand Download model → run
Time Hours per image Seconds per image

Denoising

Old Way New Way
Method Gaussian / Median filter CARE or Noise2Void
Effect on detail Destroys fine structures Preserves structures
Training needed No Yes (CARE) / No (N2V with self-training)
Fiji command run("Gaussian Blur...", "sigma=2") Plugins → CSBDeep → N2V → predict
Quality metric Noise reduced but blurry Noise reduced, detail preserved

Cell Tracking

Old Way New Way
Detection LoG / DoG detector StarDist / Cellpose detector
False detections Many (requires threshold tuning) Few (AI learned what cells look like)
Tracking errors Frequent broken/switched tracks Significantly fewer
Manual correction Hours Minutes
Fiji path TrackMate → LoG Detector TrackMate → StarDist Detector

Batch Processing

Old Way New Way
Tool Fiji Macro (.ijm) Python script + Cellpose/StarDist
File formats Limited (mostly TIFF) Any (via Bio-Formats / skimage)
Output Results table in Fiji CSV / pandas DataFrame
Plotting External (Excel, Prism) matplotlib / seaborn (in same script)
Reproducibility Good (macro is deterministic) Better (script + model version + environment)

Speed (GPU Acceleration)

Old Way New Way
Gaussian Blur 3D 12 sec (CPU) 0.05 sec (CLIJ2 GPU)
3D Watershed 45 sec (CPU) 0.5 sec (CLIJ2 GPU)
Macro syntax run("Gaussian Blur...", "sigma=2") Ext.CLIJ2_gaussianBlur2D(in, out, 2, 2)

Model Training

Old Way New Way
Approach N/A (no models to train) ZeroCostDL4Mic on Google Colab
Hardware N/A Free GPU via Google Colab
Expertise N/A Follow notebook cells
Time N/A 30-120 min training

Decision Flowchart

Need to analyze microscopy images?
│
├─ Are objects well-separated, high contrast?
│  └─ YES → Classical methods still work fine
│
├─ Are objects touching / overlapping?
│  ├─ Round (nuclei)? → StarDist
│  └─ Irregular (cells)? → Cellpose
│
├─ Is the image too noisy?
│  └─ Denoise first (CARE / N2V) → then segment
│
├─ Need to track over time?
│  └─ TrackMate + StarDist/Cellpose detector
│
├─ Need something specific (organelles, membranes)?
│  └─ Check BioImage Model Zoo → DeepImageJ
│
├─ Nothing works on your data?
│  └─ Annotate with Micro-SAM → Train custom model (ZeroCostDL4Mic)
│
└─ Processing is too slow?
   └─ CLIJ2 for GPU acceleration
