Overview

LabKit bridges the gap between classical thresholding and deep learning. It lets you paint a few strokes on your image to label foreground and background, then trains a machine learning classifier (random forest) in real-time to segment the entire image. No coding, no training datasets, no GPUs needed.

Think of it as "smart thresholding" â€” you show it what you want, and it figures out the rules.

When to Use LabKit

Scenario Best Tool
Round nuclei, well-separated StarDist (pre-trained, zero effort)
Irregular cells, any shape Cellpose (pre-trained, minimal config)
Complex texture-based segmentation LabKit (your annotations teach the classifier)
Objects with unique features not in any model LabKit (custom training in minutes)
Quick exploration / prototyping LabKit (fastest way to test if segmentation is feasible)

LabKit is ideal for situations where:
- Pre-trained models don't fit your specific data
- You need pixel classification (multiple tissue types, compartments)
- You want fast, interactive results without any setup

How to Use LabKit

Step 1: Open LabKit

  1. Open your image in Fiji
  2. Plugins â†’ LabKit â†’ Open Current Image With LabKit

Step 2: Paint Labels

  • Select the brush tool
  • Choose a label class (e.g., "foreground" / "background")
  • Paint a few strokes on representative regions
  • Add more classes if needed (e.g., "nuclei", "cytoplasm", "background")

Step 3: Train

  • Click Train Classifier (or it trains automatically)
  • The classifier applies to the entire image in real-time
  • Review the result overlay

Step 4: Iterate

  • Find areas where the classifier fails
  • Paint corrections in those areas
  • Re-train
  • Repeat until satisfied

Step 5: Apply

  • Export the segmentation as a label image
  • Apply to other images in the dataset

LabKit vs. Ilastik

LabKit is similar in concept to Ilastik (another popular pixel classification tool). Key differences:

Feature LabKit Ilastik
Runs inside Fiji Yes No (standalone application)
Large image support Excellent (BigDataViewer-based) Good
3D support Yes Yes
Algorithm Random forest on image features Random forest on image features
Export to Fiji Direct Via file export
Learning curve Minimal Moderate
Start with LabKit for quick exploration, use Ilastik for complex multi-step workflows

Combining LabKit with AI Tools

LabKit works well as a quality control or post-processing step after AI segmentation:

  1. Run StarDist or Cellpose for initial segmentation
  2. Where the AI fails, use LabKit to create a correction mask
  3. Combine the results

Or use LabKit to create training data for deep learning models:
1. Segment a few images with LabKit
2. Export as label images
3. Use as training data for StarDist or Cellpose custom models
